# seitocamera-admin

Aplicació web privada per gestionar:

- factures rebudes
- factures emeses
- conciliació bancària
- integracions amb Zoho Mail, Rentman, Qonto, Banc Sabadell, pCloud i Google Drive

## Estat del projecte

Fase inicial de preparació del repositori i arquitectura.

## Stack previst

- Frontend: Next.js
- Backend: Node.js
- Base de dades: PostgreSQL
- ORM: Prisma
- Deploy: Docker + VPS Hostinger
